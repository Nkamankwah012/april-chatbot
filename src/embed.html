AirCare Widget Integration Example

This page demonstrates how to embed the AirCare widget into any website.

Easy Integration Methods

Method 1: Auto-Initialize (Simplest)

Add this script tag to your website:

<script 
    src="https://your-domain.com/widget.js" 
    data-aircare-widget
    data-theme="light"
    data-width="400px"
    data-height="750px">
</script>

Method 2: Manual Control

For more control over initialization:

<script src="https://your-domain.com/widget.js"></script>
<script>
    // Initialize when ready
    window.AirCareWidget.init({
        containerId: 'my-widget-container',
        theme: 'light',
        width: '400px',
        height: '750px'
    });
</script>

Method 3: Custom Container

Embed in a specific element:

<div id="aircare-support"></div>
<script src="https://your-domain.com/widget.js"></script>
<script>
    window.AirCareWidget.init({
        containerId: 'aircare-support',
        theme: 'light'
    });
</script>

Available Methods

  • window.AirCareWidget.init(config) - Initialize widget
  • window.AirCareWidget.show() - Show widget
  • window.AirCareWidget.hide() - Hide widget
  • window.AirCareWidget.destroy() - Remove widget

Configuration Options

  • containerId: ID of container element (default: 'aircare-widget')
  • theme: 'light' or 'dark' (default: 'light')
  • width: Widget width (default: '400px')
  • height: Widget height (default: '750px')
  • botId: Custom bot ID
  • botUrl: Custom bot URL

The widget will appear in the bottom-right corner of the page by default, or in your specified container.
